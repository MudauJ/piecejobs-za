## 2. RATINGS & REVIEWS

Add a reviews table to Supabase:

CREATE TABLE IF NOT EXISTS reviews (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  job_id UUID REFERENCES jobs(id),
  worker_id UUID REFERENCES workers(id),
  homeowner_name TEXT,
  rating INTEGER CHECK (rating >= 1 AND rating <= 5),
  comment TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

ALTER TABLE reviews DISABLE ROW LEVEL SECURITY;
GRANT ALL ON reviews TO anon, authenticated;

When a job is marked as complete:
- Show a "Rate this Worker" modal with: star rating (1-5, clickable stars), comment textarea, submit button
- On submit: insert into reviews table, update the worker's rating (average) and review_count in the workers table
- Show a success toast "Review submitted! Thank you."
