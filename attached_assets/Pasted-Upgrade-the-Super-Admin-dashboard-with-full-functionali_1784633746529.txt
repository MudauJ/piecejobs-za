- SA ID number (masked: show first 6 digits + *****)

- Registration date
- Reviews section: list of all reviews for this worker showing homeowner name, star rating, comment, date
- ID Document upload status (show "Not uploaded" for now — we will add upload later)

Action buttons on the profile modal:

- "Suspend" (amber) — sets a new column is_suspended = true (add this column to workers table)
- "Remove" (red) — deletes worker after confirmation
- "Send Message" (blue) — opens a simple text input to send an email/WhatsApp to the worker

Add is_suspended column to workers table:

Suspended workers should show an amber "Suspended" badge instead of green "Verified".

## 2. JOBS SECTION — Full Job Details

When admin clicks on a job title, open a full job detail modal showing:
- Job title, category badge, status badge
- Full description
- Budget
- Location (suburb, city)
- Posted by (homeowner name)
- Date posted
- Date hired (if applicable)
- Date completed (if applicable)
- All applications for this job:
  * Applicant name, phone, proposed rate, message, status (pending/accepted/declined)
- Payment info (if payment exists): amount paid, platform fee, payout method, status

Action buttons:
- "Remove Job" (red) — deletes job after confirmation
- "Flag as Inappropriate" (amber) — sets a new column is_flagged = true

Add is_flagged column to jobs table:
ALTER TABLE jobs ADD COLUMN IF NOT EXISTS is_flagged BOOLEAN DEFAULT false;

Flagged jobs show an amber "Flagged" badge on the jobs list.
